# Flow App Icon 创建指南

## 设计规范

### 颜色
- **主色**：#8FB9A8（鼠尾草绿）
- **背景色**：#F8F7F2（温暖白色）

### 图标设计
- **形状**：柔和的流动曲线
- **风格**：极简、现代、有机
- **尺寸**：1024x1024px（iOS/Android标准）

### 设计元素
1. **流动曲线**：象征身体的自然律动
2. **简洁**：无文字，纯图形
3. **圆角**：使用柔和的圆角，符合设计系统

## 创建步骤

### 方案A：使用设计工具（推荐）
1. 使用 Figma/Sketch/Adobe Illustrator
2. 创建 1024x1024px 的画布
3. 背景色：#F8F7F2
4. 绘制柔和的流动曲线（使用 #8FB9A8）
5. 导出为 PNG 格式

### 方案B：使用在线工具
1. 访问 https://www.canva.com 或类似工具
2. 创建 1024x1024px 设计
3. 使用上述颜色和设计规范
4. 导出为 PNG

### 方案C：使用代码生成（临时方案）
可以使用 AppIcon.tsx 组件生成，但需要转换为 PNG 图片文件。

## 文件位置
- `assets/images/icon.png` - iOS/通用图标
- `assets/images/adaptive-icon.png` - Android自适应图标
- `assets/images/splash-icon.png` - 启动画面图标

## 临时解决方案
如果暂时没有图标文件，应用会使用默认图标。建议尽快创建符合设计规范的图标。
