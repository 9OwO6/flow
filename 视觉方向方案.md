# La Le Mo 视觉方向方案

## 方案 A：极简清爽 🎨

### 设计理念
**"干净、现代、专业"** - 类似 Apple Health、Notion 的设计风格

### 颜色系统

#### 主色调
- **Primary**: `#4A90E2` (清新蓝) - 主按钮、链接、选中态
- **Secondary**: `#7B68EE` (柔和紫) - 次要操作
- **Accent**: `#50C878` (健康绿) - 成功、健康相关

#### 中性色
- **Background**: `#FAFAFA` (极浅灰) - 页面背景
- **Surface**: `#FFFFFF` (纯白) - 卡片背景
- **Surface Variant**: `#F5F5F5` (浅灰) - 次要卡片
- **Border**: `#E0E0E0` (浅灰边框) - 分割线、边框

#### 文字色
- **Text Primary**: `#1A1A1A` (深黑) - 主要文字
- **Text Secondary**: `#666666` (中灰) - 次要文字
- **Text Tertiary**: `#999999` (浅灰) - 辅助文字

#### 功能色
- **Success**: `#50C878` (健康绿)
- **Warning**: `#FFB84D` (温和橙)
- **Error**: `#FF6B6B` (柔和红)
- **Info**: `#4A90E2` (信息蓝)

### 字体层级

```typescript
Typography: {
  h1: { fontSize: 32, fontWeight: '700', lineHeight: 40 },  // 页面主标题
  h2: { fontSize: 24, fontWeight: '600', lineHeight: 32 },  // 卡片标题
  h3: { fontSize: 20, fontWeight: '600', lineHeight: 28 },  // 区块标题
  body1: { fontSize: 16, fontWeight: '400', lineHeight: 24 }, // 正文
  body2: { fontSize: 14, fontWeight: '400', lineHeight: 20 }, // 次要正文
  caption: { fontSize: 12, fontWeight: '400', lineHeight: 16 }, // 说明文字
  button: { fontSize: 16, fontWeight: '600', lineHeight: 24 }, // 按钮文字
}
```

### 间距系统

```typescript
Spacing: {
  xs: 4,   // 极小间距
  sm: 8,   // 小间距
  md: 16,  // 标准间距
  lg: 24,  // 大间距
  xl: 32,  // 超大间距
  xxl: 48, // 极大间距
}
```

### 圆角系统

```typescript
Radius: {
  none: 0,
  sm: 8,   // 小元素（标签、小按钮）
  md: 12,  // 标准（卡片、输入框）
  lg: 16,  // 大卡片
  xl: 24,  // 超大卡片
  full: 9999, // 圆形
}
```

### 阴影系统（极简）

```typescript
Elevation: {
  none: { elevation: 0, shadowOpacity: 0 },
  sm: { elevation: 1, shadowOpacity: 0.05 },   // 轻微阴影
  md: { elevation: 2, shadowOpacity: 0.08 },   // 标准阴影
  lg: { elevation: 4, shadowOpacity: 0.12 },   // 大阴影（仅用于模态框）
}
```

### 设计特点
- ✅ **纯色背景**：白色/浅灰，无渐变
- ✅ **轻微阴影**：仅用于卡片分离，不使用重阴影
- ✅ **边框优先**：使用细边框（1px）替代阴影
- ✅ **大留白**：充足的间距，呼吸感强
- ✅ **清晰层级**：通过字体大小和颜色建立层级

## 方案 B：温暖可爱 🎨

### 设计理念
**"温暖、轻松、友好"** - 类似 Bear、Day One 的设计风格

### 颜色系统

#### 主色调
- **Primary**: `#FFB84D` (温暖橙) - 主按钮、强调
- **Secondary**: `#FF9F66` (柔和橙) - 次要操作
- **Accent**: `#6BCF7F` (清新绿) - 健康、成功

#### 中性色
- **Background**: `#FFF9F0` (极浅米色) - 页面背景
- **Surface**: `#FFFFFF` (纯白) - 卡片背景
- **Surface Variant**: `#FFF5E6` (浅米色) - 次要卡片
- **Border**: `#FFE5CC` (浅橙边框) - 分割线、边框

#### 文字色
- **Text Primary**: `#2D1B0E` (深棕) - 主要文字
- **Text Secondary**: `#6B5D4F` (中棕) - 次要文字
- **Text Tertiary**: `#9B8E80` (浅棕) - 辅助文字

#### 功能色
- **Success**: `#6BCF7F` (健康绿)
- **Warning**: `#FFB84D` (温暖橙)
- **Error**: `#FF8C8C` (柔和红)
- **Info**: `#7BC8F6` (清新蓝)

### 字体层级

```typescript
Typography: {
  h1: { fontSize: 32, fontWeight: '700', lineHeight: 40 },
  h2: { fontSize: 24, fontWeight: '600', lineHeight: 32 },
  h3: { fontSize: 20, fontWeight: '600', lineHeight: 28 },
  body1: { fontSize: 16, fontWeight: '400', lineHeight: 24 },
  body2: { fontSize: 14, fontWeight: '400', lineHeight: 20 },
  caption: { fontSize: 12, fontWeight: '400', lineHeight: 16 },
  button: { fontSize: 16, fontWeight: '600', lineHeight: 24 },
}
```

### 间距系统

```typescript
Spacing: {
  xs: 4,
  sm: 8,
  md: 16,
  lg: 24,
  xl: 32,
  xxl: 48,
}
```

### 圆角系统

```typescript
Radius: {
  none: 0,
  sm: 8,
  md: 12,
  lg: 16,
  xl: 20,  // 比方案A稍大，更柔和
  full: 9999,
}
```

### 阴影系统（轻微）

```typescript
Elevation: {
  none: { elevation: 0, shadowOpacity: 0 },
  sm: { elevation: 1, shadowOpacity: 0.06, shadowColor: '#FFB84D' }, // 带暖色调
  md: { elevation: 2, shadowOpacity: 0.1, shadowColor: '#FFB84D' },
  lg: { elevation: 4, shadowOpacity: 0.15, shadowColor: '#FFB84D' },
}
```

### 设计特点
- ✅ **温暖背景**：极浅米色，不刺眼
- ✅ **轻微阴影**：带暖色调的轻微阴影
- ✅ **柔和边框**：浅橙色边框，更友好
- ✅ **适中留白**：舒适的间距
- ✅ **友好层级**：通过颜色和大小建立层级，但更柔和

## 📊 方案对比

| 特性 | 方案A：极简清爽 | 方案B：温暖可爱 |
|------|----------------|----------------|
| **背景色** | 极浅灰 (#FAFAFA) | 极浅米色 (#FFF9F0) |
| **主色** | 清新蓝 (#4A90E2) | 温暖橙 (#FFB84D) |
| **阴影** | 极轻微，灰色调 | 轻微，暖色调 |
| **边框** | 浅灰 (#E0E0E0) | 浅橙 (#FFE5CC) |
| **文字色** | 深黑 (#1A1A1A) | 深棕 (#2D1B0E) |
| **整体感觉** | 专业、现代、冷静 | 友好、温暖、轻松 |
| **适合场景** | 专业健康记录 | 轻松日常记录 |

---

## 🎯 推荐

**推荐方案 B：温暖可爱**

**理由：**
1. 符合"轻松可爱但不幼稚"的要求
2. 温暖色调更适合健康记录类应用
3. 在专业和友好之间取得平衡
4. 与便便主题更协调（不会太严肃）
