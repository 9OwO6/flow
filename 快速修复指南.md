# Expo Go 无法加载 - 快速修复

## 立即尝试这些方法：

### 方法 1: 强制使用 Expo Go 模式 ⭐
```bash
npm start -- --go
```
这会强制使用 Expo Go，即使项目有 dev-client。

### 方法 2: 使用 Tunnel 模式（推荐）
```bash
npm start -- --tunnel
```
即使手机和电脑不在同一网络也能连接。

### 方法 3: 清除缓存重启
```bash
# 停止当前服务器 (Ctrl+C)
npm start -- --clear
```

### 方法 4: 检查终端错误
查看运行 `npm start` 后的终端输出：
- 如果有红色错误，告诉我具体错误信息
- 如果显示 "Metro waiting on..." 说明服务器正常

---

## 常见情况

### 情况 1: 扫描后显示"需要开发客户端"
**解决：** 使用 `npm start -- --go`

### 情况 2: 扫描后一直"加载中"
**解决：** 
1. 检查终端是否有错误
2. 尝试 `npm start -- --clear`
3. 重启 Expo Go 应用

### 情况 3: 扫描后没有任何反应
**解决：**
1. 确保手机和电脑在同一 WiFi
2. 或使用 `npm start -- --tunnel`
3. 尝试手动输入 URL（终端显示的 exp://...）
