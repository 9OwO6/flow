## LinkedIn Project Post: Technical

I built and polished Flow / La-le-mo, a TypeScript + Expo React Native mobile app for local-first personal pattern tracking.

Technical highlights:
- Expo Router multi-screen mobile architecture
- AsyncStorage-based local records and preferences
- CSV / JSON export flows
- i18next bilingual UI resources
- Reusable design-system components and tokens
- TypeScript QA pass with `npm.cmd exec tsc -- --noEmit`

The product challenge was handling sensitive wellness data carefully: no diagnosis claims, no cloud account dependency, and summaries framed only around saved entries.

## LinkedIn Project Post: Product-Focused

Flow / La-le-mo started as a feature-rich tracker and evolved into a calmer personal pattern app. The biggest improvements were product-oriented: simplifying Home around the daily logging loop, making History actions explicit, moving privacy/export/delete controls into a Trust & Data section, and replacing medical-sounding copy with personal reflection language.

This project helped me practice mobile UX, privacy-sensitive product design, QA thinking, and iterative product polish in a real React Native codebase.
